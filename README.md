# Abstract

This research project presents an explainable and performance-optimised machine learning framework for stock market crash prediction using historical NIFTY 50 market data. The proposed system integrates ensemble learning techniques with Explainable Artificial Intelligence (XAI) methods to enhance predictive accuracy, interpretability, and robustness across varying market conditions.

The framework combines Decision Tree, Logistic Regression, Random Forest, and XGBoost classifiers within a soft-voting ensemble architecture. SHAP (SHapley Additive exPlanations) is employed to provide transparency and identify the most influential financial indicators contributing to crash prediction. The project further incorporates advanced feature engineering, class imbalance handling through SMOTE, and rolling-window validation to ensure reliable temporal generalisation.

---

# Project Objectives

- Develop a robust ensemble-based stock market crash prediction framework.
- Improve predictive performance across multiple financial market regimes.
- Integrate explainable AI techniques for transparent decision-making.
- Address severe class imbalance using SMOTE oversampling.
- Build a scalable and research-oriented financial risk analysis system.
